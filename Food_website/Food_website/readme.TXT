1 Установите XAMPP
2 Для правильного выполнения XAMPP вам нужно открыть свойства xampp-control.exe ->
compatibility -> checked run this program as an administrator
3 Откройте панель управления XAMPP и запустите apache и mysql
4 Извлеките файлы в C:\xampp\htdocs
5 Откройте ссылку localhost/phpmyadmin
6 Нажмите кнопку создать на боковой панели навигации
7 Дайте имя базы данных как (клиника) нажмите на кнопку создать
8 После создания имени базы данных нажмите кнопку импорт
9 Просмотрите файл в каталоге [Food_website/database/appoinment.sql]
10 Откройте любой браузер и введите http://localhost/Food_website
